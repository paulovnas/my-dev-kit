# Agents

Organizacao dos agentes dentro deste kit:

- `agents/from-skills`: agentes extraidos automaticamente das skills em `skills/global/*/agents`.
- `agents/custom`: agentes criados por voce para casos especificos.

Para atualizar `agents/from-skills`, rode:

```powershell
pwsh -File .\scripts\sync-global-skills.ps1
```
